# User Guide - Expenses Tracker

## Getting Started

### First Time Setup
1. The app opens with a splash screen
2. You'll be guided through the setup process
3. Add your first account (e.g., Cash, Bank Account)
4. Categories are pre-populated for income and expenses

## Main Features

### 1. Dashboard (Home Screen)
- View total balance across all accounts
- See recent transactions
- Quick access to add new transactions
- Navigate to other sections

### 2. Adding Transactions

#### Regular Income/Expense
1. Tap the "+" button or "Add Transaction"
2. Select "Income" or "Expense"
3. Enter amount
4. Select account
5. Select category
6. Choose date (defaults to today)
7. Add optional remarks
8. Tap "Save Transaction"

#### Lend Money (You Give)
1. Select "Expense"
2. Tap on "Lend Given"
3. Enter the person's name
4. Enter amount and other details
5. Save

#### Borrow Money (You Take)
1. Select "Income"
2. Tap on "Lend Taken"
3. Enter the person's name (who you borrowed from)
4. Enter amount and other details
5. Save

#### Return Money
**When you return borrowed money:**
1. Select "Expense"
2. Tap on "Lend Returned"
3. Select the person from dropdown (shows outstanding amount)
4. Enter amount you're returning
5. Save

**When someone returns money to you:**
1. Select "Income"
2. Tap on "Lend Returned"
3. Select the person from dropdown (shows outstanding amount)
4. Enter amount they're returning
5. Save

### 3. Lend & Borrow Screen

#### I Gave (Lend) Tab
- See all people you lent money to
- View total amount to receive
- Tap on a person to see:
  - Complete transaction history
  - Total lent
  - Total returned
  - Outstanding balance

#### I Took (Borrow) Tab
- See all people you borrowed from
- View total amount to pay back
- Tap on a person to see detailed history

### 4. Reports & Analytics

#### Viewing Reports
1. Navigate to Reports screen
2. Default shows "This Month" data
3. Tap the period dropdown to change:
   - This Month
   - Last Month
   - This Year
   - Custom (select your own date range)

#### Available Insights
- Net Balance (Savings or Loss)
- Savings Rate (%)
- Average Daily Income
- Average Daily Expense
- Total Transactions
- Average Transaction Size
- Expenses by Category (pie chart)
- Category breakdown with percentages

### 5. Settings

#### Manage Accounts
- Add new accounts
- Edit existing accounts
- Delete accounts
- View account balances

#### Manage Categories
- Add custom income/expense categories
- Delete categories
- Organized by Income/Expense tabs

#### Manage Spending Limits
- Set daily, weekly, or monthly limits
- Track spending against limits
- Visual progress indicators

#### Other Settings
- Toggle Dark Mode
- Reset All Data (with confirmation)

## Tips and Best Practices

### For Lend/Borrow Tracking
1. **Always use the person's full name** - Helps avoid confusion
2. **Add remarks** - Note the purpose or context
3. **Regular updates** - Record returns promptly
4. **Check outstanding** - Review regularly in Lend & Borrow screen

### For Budget Management
1. **Set realistic limits** - Use spending limits feature
2. **Categorize properly** - Helps with reports
3. **Check reports weekly** - Stay informed
4. **Use custom dates** - Analyze specific periods

### For Accurate Tracking
1. **Record immediately** - Don't wait
2. **Be consistent** - Use same categories
3. **Include all transactions** - Even small ones matter
4. **Review regularly** - Check dashboard and reports

## Understanding the Numbers

### In Lend & Borrow
- **Total to Receive**: Money people owe you
- **Total to Pay**: Money you owe others
- **Outstanding**: Lent minus returned (per person)

### In Reports
- **Net Balance**: Income - Expense
- **Savings Rate**: (Income - Expense) / Income Ã— 100
- **Avg Daily**: Total divided by period days

## Troubleshooting

### App not loading
- Close and reopen the app
- Check for sufficient storage space
- Clear app cache if needed

### Transactions not showing
- Check date filter in Reports
- Ensure transaction was saved successfully
- Refresh the screen

### Wrong balance
- Verify all transactions are correct
- Check for duplicate entries
- Review lend/borrow transactions
